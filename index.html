Estimate Protest Potential

This runs a rough “what might be supportable” value from county land and building (improvement) amounts, then compares it to the assessed value you’re protesting to suggest a possible reduction range. Use it after you open a property from Find Property. Filling in the Notice section above is optional; it only adds context to the write-up, not to this core math.

Only affects the improvement (structure) portion of the model, not land. Increase this if the home has real problems you could argue at a hearing—neglected systems, damage, or heavy wear. 0% = no extra discount. County data often assumes “average” condition; this nudges the model when yours is below that.

Leave empty to use the assessed / market total from the county lookup. Fill this in if that number is missing in the app or you want the estimate to use the same total as on your Notice of Appraised Value (so “possible reduction” matches the value you’re actually protesting).

Protest Playbook

            
    
              
  1. File by May 15 or 30 days after your notice date (whichever is later).
    2. 
              
  2. Review BCAD details for errors (sqft, lot size, year built, features).
    3. 
              
  3. Take "ugly" photos now: foundation cracks, drainage issues, roof wear, and aging systems.
    4. 
              
  4. Collect condition evidence (photos + contractor repair estimates) for protest support, not listing prep.
    5. 
              
  5. Watch for the land value trap: if total value is flat but land value jumps, protest it immediately.
    6. 
              
  6. Protest every year, either DIY or with a protest firm.
    7. 
              
  7. Request BCAD comp packet and identify mismatches.
